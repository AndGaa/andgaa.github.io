Approval App

Internal web application for managing approval workflows

  • Role: Full-Stack Developer
  • Stack: React, Node.js, PostgreSQL
  • Year: 2024

Overview

This project was built to streamline approval workflows across teams, replacing email-based processes with a centralized web platform.

Key Features

  • Role-based access control
  • Real-time approval status
  • Audit logging

Challenges & Solutions

The biggest challenge was ensuring data consistency across concurrent approvals, which was solved using transactional updates and webhooks.
